PowerOJ is a online judge for ACM-ICPC or OI. Online Judge is a web service which can compile user's source code and run in limition.
Judge use predefined data files as stdin and get user's stdout as files, compare with the standard output data to judge the user's solution.

First of all, read carefully Frequently asked Questions.
Then, choose problem, solve it and submit it.

If you want to publish your problems or setup your own online contest, just write us
